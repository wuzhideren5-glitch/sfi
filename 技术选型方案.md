# 求职AI智能体 - 技术选型方案

> **日期**：2026-05-16  
> **前提**：港中深商科研究生 Web 应用，聊天优先，需兼容中国网络环境

---

## 一、总览

| 层级 | 选型 | 理由 |
|------|------|------|
| 前端 | **Next.js 14 (App Router)** | React 生态、SSR 可选、Vercel 一键部署 |
| 后端 | **Python FastAPI** | AI 生态最佳、async 原生、Pydantic 验证 |
| 数据库 | **PostgreSQL + pgvector** | 一库多用，免维护独立向量库 |
| AI 模型 | **GPT-4o（主）+ DeepSeek-V3（备）** | GPT-4o 解析最强；DeepSeek 便宜，国内直连 |
| Embedding | **text-embedding-3-large** | OpenAI 最强中文 embedding |
| 文件解析 | **PyMuPDF + python-docx** | 轻量、本地跑、无第三方 API 费用 |
| 部署 | **Vercel（前端）+ Railway（后端）** | 零运维、自动扩缩、免费额度够 MVP |
| 对象存储 | **Cloudflare R2** | 简历/文档存储，免费 10GB，无出口费 |

## 二、关键决策详解

### 2.1 向量数据库：PostgreSQL + pgvector（选这个）

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| **pgvector** | 和业务库同一个 DB，零运维；支持 HNSW 索引；免费 | 大规模（>10M 向量）不如专用库 | ✅ MVP 选它 |
| Qdrant | 专业向量库，性能最优 | 多一个服务要维护，部署成本 +1 | 以后数据量大了迁 |
| Milvus | 分布式，亿级向量 | 过重，MVP 杀鸡用牛刀 | ❌ |
| Pinecone | 全托管，零运维 | 贵，中国网络可能不稳定 | ❌ |

> **决策**：pgvector。画像/校友/对话三张表都在 PostgreSQL，向量字段直接加列。一条 SQL 跑混合查询（结构化过滤 + 向量相似度），无需跨服务。

### 2.2 AI 模型：GPT-4o 主力 + DeepSeek-V3 降级

| 场景 | 模型 | 原因 |
|------|------|------|
| 简历解析（PDF→JSON） | **GPT-4o** | 复杂文档结构化提取，准确率 >90% |
| 校友文档解析（Word→JSON） | **GPT-4o** | 同上，准度优先 |
| 日常对话 | **GPT-4o**（优先）/ **DeepSeek-V3**（降级） | GPT-4o 贵但体验好；高峰期/预算紧张切 DeepSeek |
| LLM-as-Judge 质检 | **DeepSeek-V3** | 独立模型，避免自己审自己的盲区，便宜 |
| Embedding | **text-embedding-3-large** | 中文效果最好，1536维 |

**成本估算（单用户 / 月）**：

| 操作 | 调用次数 | 每次 Token | 月成本 |
|------|---------|-----------|--------|
| 简历解析 | 1 次 | ~3000 token | $0.03 |
| 日常对话 | ~20 轮 | ~500 token/轮 | $0.10 |
| 画像更新 | ~5 次 | ~1000 token | $0.03 |
| 对话 Embedding | ~20 次 | ~500 token | $0.002 |
| **合计 / 用户 / 月** | | | **≈ $0.16** |

> 100 个活跃用户 → 月成本 ≈ $16。就业中心预算完全扛得住。

### 2.3 前端：Next.js 14 App Router

**页面结构**：
```
/                    → 聊天主页（需登录）
/chat/[session_id]   → 对话历史回看
/admin               → 后台管理（文档上传/校友管理/质量看板）
```

**组件树**：
```
Layout
├── ProfilePanel（左侧画像，300px 固定宽）
│   ├── UserIdentity（头像 + 姓名 + 基本信息）
│   ├── ProfileFields（教育/实习/技能/目标/差距）
│   └── UploadButton（上传简历）
└── ChatPanel（右侧聊天）
    ├── ChatHeader（AI 头像 + 状态）
    ├── MessageList（消息流）
    │   ├── WelcomeCard（首次引导）
    │   ├── MatchCard（校友匹配结果卡片）
    │   └── GapReport（差距分析报告）
    └── ChatInput（输入框 + 快捷操作 chips）
```

**UI 库选择**：
- ❌ 不用组件库（Ant Design 太重，Tailwind 够用）
- ✅ **Tailwind CSS** + 自定义 CSS 变量（保持紫色/金色主题）
- ✅ **Framer Motion**：消息动画（可选，Phase 1 可用 CSS）

### 2.5 数据库 Schema 设计

**三张独立表**（按解耦规则，不同 Schema 或前缀隔离）：

```sql
-- profile.user_profiles
CREATE TABLE profile.user_profiles (
    user_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    profile_json JSONB NOT NULL,        -- 完整画像 JSON
    embedding vector(1536),             -- 画像向量（匹配用）
    created_at TIMESTAMPTZ DEFAULT now(),
    updated_at TIMESTAMPTZ DEFAULT now()
);

-- kb.alumni_records
CREATE TABLE kb.alumni_records (
    alumni_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    source_doc VARCHAR(500),            -- 来源文档名
    record_json JSONB NOT NULL,         -- 完整校友记录 JSON
    embedding vector(1536),             -- 校友向量
    tags TEXT[],                        -- 标签数组（检索加速）
    created_at TIMESTAMPTZ DEFAULT now()
);
CREATE INDEX idx_alumni_embedding ON kb.alumni_records 
    USING hnsw (embedding vector_cosine_ops);

-- chat.conversations
CREATE TABLE chat.conversations (
    session_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL,
    messages JSONB NOT NULL,            -- [{role, content, timestamp}]
    embedding vector(1536),             -- 整段对话向量（记忆检索用）
    created_at TIMESTAMPTZ DEFAULT now()
);
```

### 2.6 认证方案（MVP 阶段）

### 2.7 文件存储策略

| 文件类型 | 存储位置 | 说明 |
|---------|---------|------|
| 学生简历 PDF | Cloudflare R2（加密） | 解析后原文件保留，用户可随时删除 |
| 校友访谈 Word | Cloudflare R2 | 后台管理员上传，解析后保留原件 |
| 对话记录 | PostgreSQL JSONB | 不需要文件存储 |
| Embedding 向量 | pgvector | 和结构化数据同库 |

## 三、开发环境搭建清单

### 3.1 本地开发

```bash
# 前端
npx create-next-app@latest frontend --typescript --tailwind --app
cd frontend && npm run dev        # → localhost:3000

# 后端
python -m venv venv && source venv/bin/activate
pip install fastapi uvicorn sqlalchemy asyncpg pgvector
pip install pymupdf python-docx openai
uvicorn main:app --reload         # → localhost:8000

# 数据库
brew install postgresql@16
createdb career_ai
psql career_ai -c "CREATE EXTENSION vector;"
```
