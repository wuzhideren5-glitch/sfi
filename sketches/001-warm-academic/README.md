## Variant: Warm Academic（温暖学院风）

### Design stance
用柔和的紫色调 + 金色点缀营造「校园温暖感」，像一位亲切的学长学姐在帮你。

### Key choices
- Layout: 左侧白色画像面板 + 右侧浅紫灰对话区
- Typography: Noto Sans SC，圆润亲和
- Color: 浅紫背景 #FAF8FC，紫色 #6B2D8E + 金色 #C8962E 点缀
- Interaction: 圆角气泡、卡片式画像、柔和阴影、fadeIn 动画

### Trade-offs
- Strong at: 亲切感、学生友好、第一眼不压迫
- Weak at: 信息密度偏低，专业感不如深色方案

### Best for
首次使用的学生，降低心理门槛，让人愿意开始聊天
