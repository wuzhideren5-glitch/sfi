## Variant: Professional Academic（专业学术风）

### Design stance
深色画像面板 + 白色对话区，像高端金融终端与 ChatGPT 的结合，传递「专业可信赖」。

### Key choices
- Layout: 左侧深紫黑画像面板（暗色）+ 右侧白底对话区
- Typography: Inter + Noto Sans SC，更锐利、更专业
- Color: 深色底 #1E1330，金色主按钮，高对比
- Interaction: 精准阴影、slipUp 弹性动画、悬停高亮、result-card 交互

### Trade-offs
- Strong at: 专业感强、信息层次清晰、画像可读性好
- Weak at: 深色面板可能让部分学生觉得「太正式」

### Best for
有一定求职经验的学生，或者习惯专业工具的用户
